# Social Post Template

## Structure

```
[HOOK] — First line grabs attention (question, surprising fact, relatable problem)

[BODY] — Value delivered (tip, insight, story, before/after)

[CTA] — Natural call to action (not salesy)

[HASHTAGS] — Platform-appropriate (2-5 max)
```

## Platform Variants

### Facebook (100-200 words)
- Conversational, slightly longer
- Ask a question to encourage comments
- Can include a link
- Example hook: "Ever had that moment when..."

### Instagram (50-150 words)
- Visual-first — suggest an image concept
- Line breaks for readability
- Hashtags in a separate block (after dots)
- Example hook: "Swipe to see the before and after 👆"

### Twitter (under 280 chars)
- One punchy thought
- 1-2 hashtags max
- Example hook: "Quick tip for [season]:"

### LinkedIn (100-300 words)
- Professional but personable
- Share expertise or insight
- No emoji overuse
- Example hook: "Something I've learned in [X] years..."

## Tone Checklist
- [ ] Sounds like the client, not a marketing agency
- [ ] Genuinely helpful, not just promotional
- [ ] Local references feel natural
- [ ] CTA is soft — "give us a call" not "BUY NOW"
- [ ] No avoid words from client config
- [ ] Would you actually stop scrolling for this?

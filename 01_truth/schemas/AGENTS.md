# Partner instructions (schemas)

## Scope

Applies when drafting or updating `01_truth/schemas/*`.

## Truth-bar

- Schemas are **pinhole tight**. If \(1 \neq 1\), treat it as a defect and fix upstream assumptions.

## Rules

- Treat schema docs as **reference**: precise, testable, versioned.
- No invented fields. If a field is desired but not evidenced, mark `[LOGIC TO BE CONFIRMED]`.
- Record provenance for every schema decision (what source/process requires it).
